# Comparative Financial Statement Analysis of Engro Fertilizers Limited

## Overview

This project presents a comparative financial statement analysis of Engro Fertilizers Limited (EFERT) using financial ratios, common-size analysis, horizontal analysis, and risk-return assessment.

The analysis compares EFERT with Fauji Fertilizer Company (FFC) to evaluate liquidity, profitability, leverage, efficiency, and financial performance.
